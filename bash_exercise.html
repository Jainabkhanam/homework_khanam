The $HOME environment variable stores the path to the home directory. By using echo $HOME, we can display this path. The ls ~ command lists all files and folders in the home directory, allowing us to observe its contents. Exercise 2: $cd ~/../../.. Solution: The command cd ~/../../.. takes up three directories from the current location. $cd Desktop/git/files Explanation: The cd command is used to change directories. In this case, ~/../../.. moves up three directories from the current location. The tilde (~) represents the home directory, and each ../ moves up one directory level. By combining these, we navigate to a higher-level directory.
